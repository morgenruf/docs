# Morgenruf Docs

Documentation site for [Morgenruf](https://morgenruf.dev), deployed at [docs.morgenruf.dev](https://docs.morgenruf.dev).

## Structure
- `index.html` — Home / overview
- `getting-started.html` — Installation guide
- `configuration.html` — Configuration reference
- `self-hosting.html` — Self-hosting guide (k8s, Docker)
- `helm.html` — Helm chart reference
- `faq.html` — Frequently asked questions

## Local development
Open any `.html` file directly in your browser, or use:
```bash
python3 -m http.server 8080
```
