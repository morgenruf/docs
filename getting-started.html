Getting Started

Add Morgenruf to your Slack workspace and run your first standup in minutes.

Hosted Version (Easiest)

The fastest way to get started is with the hosted version — nothing to install, nothing to configure beyond a few DM commands.

  1. Add Morgenruf to Slack

    Click the button below to start the Slack OAuth flow. You'll be asked to authorize Morgenruf to send messages and read channel info.

    Add to Slack

  2. Authorize the app

    Slack will show you an OAuth permission screen listing what Morgenruf needs. Review the scopes and click Allow. You'll be redirected back to Slack.

  3. Done — Morgenruf will DM you

    Within a few seconds, Morgenruf sends you a welcome DM. It will guide you through setting your standup channel, schedule, and timezone.

ℹ️

Only the person who installs Morgenruf needs to go through the OAuth flow. Other team members are added by inviting Morgenruf to your standup channel.

What Happens Next

After installation, Morgenruf will DM the installing user to walk through initial setup:

  • Confirm which channel standup summaries should be posted to
  • Set the time and timezone for daily standup reminders
  • Choose which days of the week to run standups (default: Mon–Fri)
  • Optionally customize the three standup questions

You can change any of these settings later by DMing Morgenruf — see the Configuration page for all available commands.

Invite Morgenruf to Your Channel

To enable standup summaries in a channel, invite Morgenruf to it:

/invite @Morgenruf
slack

Run this command inside the channel where you want summaries posted. Morgenruf also needs to be in the channel to DM the members (it can see channel membership for channels it's a member of).

Running Your First Standup

Standups run automatically at the configured time. But you can trigger one immediately for testing:

@Morgenruf standup
slack

Morgenruf will DM every member of the standup channel asking the three questions. Once everyone responds (or after the collection window closes), it posts the summary to the configured channel.

💡

The collection window defaults to 2 hours. Responses submitted after that window are still saved but may not appear in that day's summary.
